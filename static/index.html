Chart Difficulty Predictor

Welcome! This tool will help estimate a difficulty rating for your Stepmania charts.

It uses a machine learning model to predict the numerical difficulty based off several features that were extracted using this parser.
Huge shoutout to Tim Murphy for building the parser and making this project possible!

For a breakdown of methods, check out the Github repo.

Accepted file formats: .sm, .ssc.
Uploading multiple files is accepted.
Specify whether the charts are stamina or tech (this changes which model is used to make a prediction).
Ratings are based on ITG difficulty.
Currently cannot handle doubles charts.
